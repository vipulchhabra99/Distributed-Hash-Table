**Installing Dependencies**

It is assumed that the `Python3+` version is being used for evaluation.
Flask

    pip3 install flask

**Execution of Code**

To execute the server use -

    python3 server.py port

To attach to a ring

    python3 server.py port -r ring_port

To execute the client use -

    python3 client.py
